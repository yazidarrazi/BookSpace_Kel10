BOOKSPACE - Sistem Perbelanjaan Online Berbasis Java

Deskripsi Singkat Proyek
BookSpace adalah sistem toko buku online sederhana yang dikembangkan menggunakan Java dengan menerapkan konsep-konsep Pemrograman Berorientasi Objek. Sistem ini memanfaatkan encapsulation untuk melindungi data pengguna dan transaksi, inheritance dalam mengelola hierarki admin dan customer, polymorphism pada fitur keranjang belanja dan metode pembayaran, serta abstraction untuk menyembunyikan kompleksitas sistem. Penerapan collection membantu mengelola katalog buku secara efisien, sementara exception handling memastikan sistem dapat menangani kesalahan input dengan baik.
